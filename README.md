# DSList

API REST desenvolvida em Java com Spring Boot para gerenciamento de listas de jogos.

## Tecnologias
- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven

## Funcionalidades
- Listagem de jogos
- Consulta por ID
- Listagem por categorias
- Reordenação dos jogos na lista

## Objetivo
Praticar desenvolvimento de APIs REST, arquitetura em camadas, JPA e manipulação de banco de dados.
